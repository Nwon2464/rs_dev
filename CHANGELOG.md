# 변경 로그

이 문서는 사용자에게 영향을 주는 주요 변경 사항을 기록합니다.

## 2026-07-16

### 게임 데이터 변경

- 비규격 피리의 4번째 모조 개방 옵션에서 `동반자의 물리 더블 크리티컬 대미지 증가` 2티어 수치를 11%에서 14%로 변경했습니다.
- 비규격 관의 4번째 불타는 개방 옵션에 `마법 치명타 확률 증가`를 추가했습니다. 2·3·4티어 수치는 각각 14%, 17%, 20%입니다.
- 비규격 관의 4번째 불타는 개방 옵션에서 `마법 치명타 대미지 증가`의 2·3·4티어 수치를 14%·17%·20%에서 10%·12%·14%로 변경했습니다.
- 비규격 관의 4번째 불타는 개방 옵션에서 `마법 강타 확률 증가`의 2·3·4티어 수치를 10%·12%·14%에서 9%·11%·13%로 변경했습니다.
- 비규격 관의 4번째 불타는 개방 옵션에서 `마법 강타 대미지 증가`의 2·3·4티어 수치를 9%·11%·13%에서 5%·6%·8%로 변경했습니다.
- 위 비규격 관 옵션들의 티어별 출현 확률은 기존과 동일한 8.33%·2.11%·0.67%로 유지됩니다.

## Unreleased — 2026-07-13

### 추가

- 결과가 없을 때 필터를 초기화할 수 있는 버튼을 추가했습니다.
- 일반 장비 개방 옵션의 변환 확률에 간단한 안내를 추가했습니다.
- 현재 위치를 선택 조건별 칩으로 표시해 탐색 상태를 쉽게 확인할 수 있도록 했습니다.
- 개방 줄을 모두 해제했을 때 `개방 줄 미선택` 상태를 표시하고, 별도 `선택 해제` 버튼을 추가했습니다.

### 변경

- 처음 방문하는 사용자의 기본 언어를 일본어로 변경했습니다. 기존에 저장된 언어 선택은 유지됩니다.
- 일반 장비 및 비규격 장비 홈·탐색 화면의 한국어와 일본어 문구를 사용자 친화적으로 다듬었습니다.
- `개방옵션변환기改`의 한국어 표시명을 `개량된 개방옵션변환기`로 변경했습니다.
- 일반 장비와 비규격 개방옵션의 개방 줄 선택 UI 및 전체 선택 동작을 통일했습니다.
- 아이템 등급 표기에서 내부 숫자 코드(7·8·9)를 제거했습니다.

### 개선

- 선택된 언어와 필터 칩의 색상·테두리 상태를 정리해 선택 여부를 더 쉽게 구분할 수 있도록 했습니다.
- 태그 필터의 펼침·접힘 화살표를 현재 상태에 맞는 하나의 아이콘만 표시하도록 개선했습니다.
